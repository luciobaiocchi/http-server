About This Project

This project demonstrates how to build a basic HTTP server using Python sockets. The server supports MIME type detection, logging, 404 handling, and serves a fully responsive frontend.

Author

Lucio Baiocchi is a student passionate about full-stack development and embedded systems. He created this project as an educational exercise in network programming and frontend design.
