About

Scopri il progetto e l'autore dietro MyMinimalSite.

Il progetto

Server HTTP minimale con Python socket, logging, supporto MIME e integrazione Bootstrap per uno styling rapido.

L'autore

Lucio Baiocchi, sviluppatore full-stack appassionato di tecnologie moderne, Docker, IoT e design web.

Mission

Fornire un esempio semplice e modulare per comprendere il funzionamento di un web server senza dipendenze complesse.
