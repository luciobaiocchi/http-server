GymLife

Body builder simulation

Project description

Gymlife is a videgame that simulates a bodybuilder's life. The main goal of the player is to max out his muscolar mass level in each muscolar group (leg, chest, back) balancing tre key stats: Stamina, Humor and muscolar mass.

The monetary, needed to buy food and to have access to the gym, is maganed trought a minigame and also the gym includes other various minigames. The player is provoded with an inventory in wich he can store food and manage his resources.

This project is realized for the final exam of the Object oriented programation couse, in collaboration with a collegue's team.

Tecnologies used:

Java for the programming part in particular using an MVC architecture, JUnit for testing anch Gradle to manage dependecies.
